Detailed Description

Shader creator tool. Loads and compiles a shader, either from a file or from code directly. After compilation holds OpenGL id. Is used to create a ShaderProgram.

Note
This resource is not ment to be used as a std::shared_ptr and has no central counter. You just create it, use it, and once you don't need it delete it.
Warning
If you try to copy this class somewhere, the deconstructor will be called and the shader will no longer exist on the GPU thus the copy will be useless, even harmful.
